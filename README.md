# Temperature Converter

A simple and intuitive temperature conversion app that allows users to convert between different temperature units.

# About the App

This app is built using Android Studio with Java and XML. It provides a clean and user-friendly interface for converting temperatures between the following units:
- Celsius (°C)
- Fahrenheit (°F)

# Features
- Convert temperatures between Celsius,and Fahrenheit. 
- Supports both integer and decimal temperature values
- Easy-to-use interface with a simple input field and conversion buttons

# Technical Details

- Built using Android Studio with Java and XML
- Utilizes a simple and efficient conversion algorithm
